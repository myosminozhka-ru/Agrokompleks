Условия
сотрудничества
1
Регистрация
Наш менеджер свяжется с Вами для уточнения деталей и подъедет в удобное для Вас время и место для подписания договора
2
Выберите товар
Выбирая из представленных разделов интересующие позиции, добавляйте их в корзину с помощью кнопки «В корзину»
3
Оформите заказ
Для продолжения оформления заказа необходимо перейти в "Корзину". После проверки Вы можете изменить количество товара или убрать лишнее. Далее нажмите кнопку "Оформить заказ"
4
Выберите адрес и время доставки
Введите адрес для доставки и укажите удобное для Вас время из доступного в вашем регионе
5
Оплатите заказ
Выберите удобный для Вас способ оплаты заказа из предложенных
